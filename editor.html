Publish settings

To publish with one click, paste a GitHub token. It is stored only in this browser (localStorage) and sent only to GitHub. You can always use “Download” instead and upload manually.

Create one at github.com → Settings → Developer settings → Fine-grained tokens. Repository access: only perspectivelabs-site. Permission: Contents → Read and write.
